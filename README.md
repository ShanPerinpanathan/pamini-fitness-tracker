# Pamini's Blueprint Tracker v2

## Features
- **Home** — Daily dashboard: calorie rings, deficit summary, workout progress, week strip, step goal
- **Workout** — OTF class info + supplemental exercises with photos, set/rep logger, liftmanual.com links
- **Food** — Meal checklists (auto workout/rest day), live macro tracking, **add custom foods**
- **Supps** — Morning / pre-workout / night with checkboxes + Amazon links
- **Bonus** — Extra activity tracker with calorie burn (affects net deficit on Home). Add custom activities.
- **Progress** — Weight tab (chart, history, loss tracking) + Body Measurements tab (waist, hips, chest, arms, thighs, BF%)

## Deploy
1. Upload this folder to GitHub (see steps)
2. Connect GitHub repo to Vercel
3. Add env vars if using Supabase
4. Deploy → Add to iPhone home screen via Safari Share → Add to Home Screen
